Tag escolhida: UL e LI

O que ela faz?

As tags "UL" (Unordered List) e "LI" (List Item) são usadas para criar listas não ordenadas.

O "ul" é o contêiner que agrupa os itens, indicando que a ordem não é conceitualmente importante (aparece com marcadores, como bullet points). O "li" define cada item individual dentro dessa lista. Ambas trabalham juntas para dar estrutura semântica a coleções de informações.

Qual o motivo da escolha?

UL:

Motivo da Escolha: Serve como um contêiner que informa ao navegador (e a leitores de tela para acessibilidade) que o conteúdo ali dentro é uma lista, mas que a sequência em que os itens aparecem não altera o significado (ex: uma lista de compras ou uma lista de hobbies).

LI:

Motivo da Escolha: Representa cada item individual dentro do contêiner ("ul" ou "ol"). É essencial para a estrutura, pois cada pedaço de informação da lista deve ser encapsulado por um "li".

Qual a importância dela?

UL:

A tag "ul" (unordered list) é importante no HTML por sua semântica, acessibilidade e organização do conteúdo. Ela serve para estruturar listas de itens que não precisam de uma ordem específica, distinguindo-se da tag "ol" (ordered list), usada para listas sequenciais.

LI:

A tag "li" é crucial para estruturar listas em HTML, pois define cada item individualmente e garante que as listas sejam semanticamente corretas, acessíveis e bem organizadas.
